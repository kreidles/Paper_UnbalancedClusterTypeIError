Summary of Statistical Methods

The table below summarizes each of the ten statistical methods tested. One-stage models are linear mixed models, which account for within-cluster correlation using a random intercept. Two-stage models first calculate the average outcome within each cluster, and the analyze the resulting cluster means using a general linear model.
